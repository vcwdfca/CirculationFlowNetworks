# Circulation Flow Networks（环流网络）

一个 Minecraft 能源网络模组，支持 1.12.2 / 1.20.1 / 1.21.1，通过节点与中枢构建跨距离、跨维度的能源传输网络。

---

## 概述

还记得《戴森球计划》里用无线电塔把整颗星球的能源串联起来的感觉吗？这个模组的灵感就来自于此——Minecraft里也应该有一种不用铺满线缆、放下节点就能自动组网的能源系统。

环流网络允许玩家在世界中放置各类节点，组成网状的能源传输网络。网络以**中枢**为核心，连接周围的**端口节点**、**充能节点**和**中继节点**，自动检测附近的能源机器并进行能源的采集与分配。不需要手动布线，节点之间只要在链接范围内就会自动连接——有点像先铺电线杆再接用电器，但省掉了中间那堆电线。

支持 Forge Energy (FE)、IC2 EU（仅 1.12.2）等能源体系，并提供 API 允许其他模组注册自定义能源处理器。

---

## 方块

### 节点基座

节点的放置基础。端口节点、充能节点和中继节点必须放置在节点基座上方，破坏基座会导致上方的节点掉落。

### 端口节点

能源网络的基本单元。检测自身周围一定范围内的能源机器（发电机、用电器、储能设备），并将它们纳入网络。

- 能源检测范围：8 格
- 链接范围：12 格

### 充能节点

为范围内的玩家身上的物品充能。需要通过中枢的充能配置界面指定充能的装备栏位。

- 充能范围：5 格
- 链接范围：8 格

### 中继节点

不参与能源检测，仅用于延伸网络的链接距离。

- 链接范围：20 格

### 中枢

网络的控制中心。每个网络只能有一个中枢。提供 GUI 界面进行网络管理，支持插件系统、频道系统和权限管理。

- 能源检测范围：10 格
- 链接范围：16 格
- 插件槽位：5 个

### 环流屏蔽仪

在指定范围内阻止能源流动，支持红石信号控制：

- **常规模式**：收到红石信号时阻止能源流动
- **反转模式**：收到红石信号时允许能源流动

可在 GUI 中切换模式并开关范围可视化。

## 物品

### 环流配置器

多功能检查与配置工具，具有两个主模式：

**网络信息显示模式** — 潜行 + 右键节点以查看信息

- 显示全部信息
- 显示连接范围
- 显示节点网络

**能源节点配置模式** — 潜行 + 右键机器以手动覆盖类型检测

- 能量输出（发电机）
- 能量输入（用电器）
- 能量存储
- 清除配置

操作方式：潜行 + 右键空气切换主模式，潜行 + 滚轮切换子模式。

### 中枢频道插件

插入中枢后可将网络连接到指定频道。频道信息存储在物品 NBT 中，转移频道插件也可以使中枢绑定到频道插件上的频道。

### 广域充能插件

使中枢所在网络能够为同一维度内所有具有权限的玩家充能。

### 维度充能插件

使中枢所在网络能够跨维度为所有具有权限的玩家充能。

## 中枢界面

中枢提供多个功能标签页：

| 标签   | 功能                       |
|------|--------------------------|
| 节点列表 | 查看网络中所有节点，双击可定位到世界坐标     |
| 充能配置 | 选择哪些玩家接收充能，按装备栏位分类控制     |
| 插件列表 | 管理中枢的 5 个插件槽位            |
| 频道列表 | 浏览和切换可用频道                |
| 权限设置 | 管理频道成员角色（所有者 / 管理员 / 成员） |
| 频道设置 | 创建与删除频道                  |
| 能源显示 | 实时查看网络能源状态               |

---

## 频道与权限

网络通过**频道**隔离。频道分为三种可见性：公开、团队、私有。

每个频道具有权限系统：

- **所有者**：频道创建者，拥有全部权限，不可转移
- **管理员**：可管理网络配置和成员
- **成员**：可使用网络

---

## 网络连接机制

每个节点具有**链接范围**属性，处于彼此链接范围内的节点自动建立连接，组成网状拓扑。链接类型包括双向连接和单向连接。

能源检测节点（端口节点、充能节点、中枢）会在自身的**能源范围**内扫描能源机器，并自动识别机器类型（发电机 / 用电器 /
储能设备）。识别有误时可通过环流配置器手动覆盖。

同一网络不允许存在多个中枢，放置时会检测冲突。

---

## 配置

所有节点的范围参数均可在配置文件中调整。此外可配置：

- 能源机器黑名单（按类名前缀排除不兼容的 TileEntity）
- 供应操作黑名单
